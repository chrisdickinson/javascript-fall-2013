Lesson 5: Objects

Javascript Objects

Objects are Javascript fundamental data management structure. Most of the time you write Javascript, you'll be dealing with Objects


          var contact = {
            firstName: "David",
            lastName: "Manning",
            phone: "503-KL5-1337",
          };
          console.log(contact.firstName + " " + contact.lastName);

Objects are basically sets of key-value pairs. You can think of them like dictionaries.

Javascript Objects


          var object = {};
          object.tires = 4;
          object.color = "Green";
          object.horn  = true;

You can create an empty object, and add key/value pairs programatically as needed.

Javascript Objects


          var object = {};
          object["color"] = "orange";
          object["fluffy"] = true;
          object["paws"] = 2;

You can also create object keys dynamically using index notation.

Passing objects into functions

Objects passed into functions are passed by reference. This means the function receiving the object is working with the original object, rather than a copy

When passing a number, string or boolean value, a local copy is created with the function body 

            var aNumber = 1;

            function addOne (x) {
              x += 1;
            } 

            addOne(aNumber);

            console.log(aNumber); // prints: 1
When we pass an object into the function, we are working with the same object that exists in the outer scope. 

            var myBologna = { firstName: "O S C A R"};

            function bolognaNamer (bologna) {
              bologna.secondName = "M A Y E R";
            }

            console.log(myBologna);
            // prints: { firstName: 'O S C A R', secondName: 'M A Y E R' }

Note: This is also true for arrays passed into functions

Object methods

You can store functions into object keys. Such functions are usually called object methods


            var kittie = {
              eyes: "green",
              name: "mittens"
              speak: function () {
                console.log("Meow!");
              }
            }
            kittie.speak();
            // prints: "Meow!";

Object methods


              var person = {
                firstName: "David",
                lastName: "Manning"
                sayMyName: function () {
                  console.log(this.firstName + " " + this.lastName);
                }
              }

              person.sayMyName();

Within an object method, you can use this.property to reference one of the objects properties
