Arrays


        var dogs = ["Fido", "Rex", "Maggie", "Pumpkin"];

        console.log(dogs[2]);
        // Maggie

Javascript arrays are a way store a set of data for either random or sequential access. We refer to a particular element of the array by using an index notation, just like with individual characters in Strings.

Javascript


        var dogs = ["Fido", "Rex", "Maggie", "Pumpkin"];

        for (var i = 0; i <= dogs.length; i++) {
          console.log(dogs[i]);
        }

        // Fido
        // Rex
        // Maggie
        // Pumpkin

We can iterate through the array sequentially with a loop. Just like with Strings

Array Functions

Javascript provides a number of helpful functions for use with arrays. We'll look at a few.


            // push and pop are array methods that add
            // or remove elements from the end of an array

              var foo = [7, "eleven"];

              foo.pop(); // returns "eleven"
              console.log(foo); // prints [ 7 ]

              foo.push(11); // returns 2, the new length of the array
              console.log(foo); // prints [ 7, 11 ]

Array Functions


        // shift and unshift are like push and pop except they
        // add and remove elements from the beginning of the array

        var bar = ["circle", "K"];

        bar.shift(); // returns "circle"
        console.log(bar); // prints [ "K" ]

        bar.unshift("O"); // returns 2, the new length of the array
        console.log(bar); // prints [ "O", "K" ]

Array Functions

split is actually a String function, but it returns an array


  var someStupidString = "I am the very model of a modern major general"

  var words = someStupidString.split(" ");
  console.log(words);

  // ["I", "am", "the", "very", "model", "of", 
  //  "a", "modern". "major", "general"]

split breaks a string into an array using the specified character as the breaking deliminator.

Array Functions

Two other Array functions, reverse and join, can be combined with split to solve the word reversal problem in only one line!


var sen = "Four score and seven years ago Winne-the-Pooh was first released!"

var reversedSen = str.split(" ").reverse().join(" ");

console.log(reversedSen);

// prints: 'released! first was Winne-the-Pooh ago years seven and score Four'

// array.reverse() reverses the elements of the array returned by split()
// array.join(" ") joins the elements of the array into a new String,
// separated by a " "

It's important to realize that this sequence of operations does not affect the original value of the string.
