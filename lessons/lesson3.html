Writing your own functions

Writing functions

A function declaration statement looks like this:


            function myAwesomeFunction (param1, param2, etc...) {
              // here's where we put some code to do things

              return someValueWeHaveComputed;
              // return statements are optional,
              // but you need them if you want to use your
              // function call as an expression
            }

Then we can call our function like this:


            myAwesomeFunction(param1, param2);
            // if we don't need or didn't specify a return value, or
            var aValue = myAwesomeFunction(param1, param2);
            // if we did

Writing Functions

Example


              function toThePowerOf(base, exponent) {
                var result = 1;

                for (var i = 1; i <= exponent; i += 1) {
                  result *= base;
                }

                return result;
              }

              console.log(toThePowerOf(2, 8));
              // will print out 256
